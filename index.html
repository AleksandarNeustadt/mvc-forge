Asset dev server

This URL is only for Vite while npm run dev runs. Open the real site via your PHP URL. For HMR with APP_DEBUG=true, set VITE_DEV_SERVER_ORIGIN in app/.env to the exact origin Vite prints (include port), e.g. http://localhost:5137 or, if the PHP site is HTTPS and the browser blocks mixed content, run VITE_DEV_HTTPS=1 npm run dev and use https://localhost:… in VITE_DEV_SERVER_ORIGIN (accept the self-signed cert once).
